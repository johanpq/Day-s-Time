# Projeto de Hora Dinâmica

Este é um projeto simples desenvolvido utilizando HTML5, CSS3 e JavaScript. A proposta é exibir uma imagem correspondente à hora local da máquina do usuário, proporcionando uma experiência visual dinâmica ao longo do dia.

## Funcionalidades

- **Hora Dinâmica:** A imagem exibida é atualizada de acordo com a hora local do usuário.
- **Variedade de Imagens:** Diferentes imagens são associadas a diferentes períodos do dia (manhã, tarde, noite).
